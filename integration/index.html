Kettle.js

Kettle.js is a small javascript ui widget that extends drag.js to build scrollable content blocks.

Fork/Download @ Github

See github repo for Ender installation instructions.

Dependancies

  • drag.js
  • Bean
    (also required by drag)
  • move.js
    (optional for elasticity)
